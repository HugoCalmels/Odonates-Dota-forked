TEAMS :

Each registered user can create or join a team.
A team is mandatory to play Scrims.

SCRIMS :

Since CM (Captain's Mode) has been removed from Dota search options, we decided it would be cool to bring it back to the teams that needed it.
The tool includes 1v1, 2v2, 5v5, AP, RD, and CM options.
It works based on UTC times, so it doesn't matter if you are from France or Brazil.
To join a scrim, you need to send a scrim proposal to the other team. Only after the scrim proposal is accepted by the other team, then the scrim status will change to "READY".
Another tool is Scrim History, where players can see their entire scrim history, or any scrim they joined/created.

MUST KNOW :

Lobbies are not automatically created, as players will have to do this manually with the given Lobby Name and Lobby Password so all players can follow up easily.
